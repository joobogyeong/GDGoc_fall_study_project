## 1. 프로젝트 개요

### 프로젝트 필요성
- 기본적인 게시판 CRUD 구현
- 스프링 시큐리티를 통한 인증 및 인가 과정 구현
- XSS 공격 방지, CSRF 방지 등 기본적인 보안 처리 해결방법 제시


### 구현 기능
- Spring Security 구현
- API 설계
- DB 설계
- 프로젝트 환경 구성
- 게시판(CRUD) 개발
- 로그인/로그아웃 및 회원가입 기능 구현
